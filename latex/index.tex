 A C++ simulation of a load-\/balancing system that dynamically manages web servers to handle incoming requests, with firewall/\+DoS protection and real-\/time statistics tracking.\hypertarget{index_autotoc_md1}{}\doxysection{Features}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Dynamic web server scaling based on queue depth
\item \mbox{\hyperlink{structRequest}{Request}} queue with random IP generation and job types (Processing / Streaming)
\item IP range blocker and DoS prevention firewall
\item Color-\/coded console output with ANSI escape codes
\item File-\/based configuration (no need to recompile to adjust parameters)
\item Persistent log file with per-\/cycle summaries
\item Runtime statistics\+: requests generated, accepted, blocked, completed, and server scale events
\item Configurable per-\/cycle delay to control simulation speed
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{Project Structure}\label{index_autotoc_md2}

\hypertarget{index_autotoc_md3}{}\doxysection{Configuration}\label{index_autotoc_md3}
