\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classLogger}{Logger}} class and ANSI color code constants. 

The \mbox{\hyperlink{classLogger}{Logger}} writes color-\/coded messages to stdout and simultaneously writes plain text to a log file for persistent records.
