\doxysubsection{Detailed Description}
Stores all tunable parameters for the load balancer simulation. 

Default values are set in the constructor. Calling \mbox{\hyperlink{classConfig_a727645c1fe7660faad404a29bc1f13e8}{load\+From\+File()}} overrides them with values from the config file.
