\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classIPBlocker}{IPBlocker}} class that acts as a simple firewall. 

Blocks IPs that fall within a configurable range, and also blocks individual IPs that send too many requests (DOS prevention).
