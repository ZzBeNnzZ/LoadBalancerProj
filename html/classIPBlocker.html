bool IPBlocker::isBlocked ( string  ip)

Checks if a given IP address should be blocked. An IP is blocked if: 1) It falls within the blocked IP range, OR 2) It has sent more requests than the threshold.

Checks if an IP should be blocked. Also tracks how many times each IP is seen.

This method also increments the hit count for the IP.

Parameters
ipThe IP address to check.
Returns
True if the IP is blocked, false if it is allowed.
   Blocking happens if:
   1) The IP falls within the configured blocked range, OR
   2) The IP has been seen more times than the threshold.
Parameters
ipThe IP address to check.
Returns
True if blocked, false if allowed through.
